# Todo App

A simple [TodoMVC](http://todomvc.com) app to experiment with some new tech, including:

- [Create React App](https://github.com/facebookincubator/create-react-app)
- ES6
- Webpack 1.13.2
- React 15.3.2
- Redux 3.6
- Firebase 3.4

## Usage

Development:

```
npm start
```

Production Build:

```
npm run build
```

Firebase Deployment:

```
firebase use --add
firebase deploy
```
